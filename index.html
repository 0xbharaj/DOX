Data Models & Database Schema

Comprehensive data modeling for loan portfolio management

LoanPortfolioItem Model

Central data model containing all loan and vehicle information with PCAF emissions data

interface LoanPortfolioItem extends Document {
  // Loan Information
  loan_id: string;                    // Unique identifier
  borrower_name: string;              // Borrower information
  loan_amount: number;                // Original loan amount
  outstanding_balance: number;        // Current balance
  interest_rate: number;              // Annual interest rate
  term_months: number;                // Loan term
  origination_date: Date;             // Loan start date

  // Vehicle Information
  vehicle_details: {
    make: string;                     // Vehicle manufacturer
    model: string;                    // Vehicle model
    year: number;                     // Model year
    type: VehicleType;                // Car, truck, etc.
    fuel_type: FuelType;              // Gas, electric, etc.
    value_at_origination: number;     // Vehicle value
    efficiency_mpg?: number;          // Fuel efficiency
    annual_mileage?: number;          // Yearly mileage
  };

  // PCAF Emissions Data
  emissions_data: {
    annual_emissions_tco2e: number;   // Annual CO2 emissions
    attribution_factor: number;       // Loan attribution (0-1)
    financed_emissions_tco2e: number; // Attributed emissions
    data_quality_score: number;       // PCAF quality (1-5)
    pcaf_data_option: PCAFOption;     // PCAF methodology
    calculation_method: string;       // Calculation approach
    last_calculated: Date;            // Last update
  };
}

PCAF Calculation Engine

Complete implementation of PCAF Section 5.6 methodology

Core PCAF Formula

Financed Emissions = Annual Vehicle Emissions √ó Attribution Factor
Where: Attribution Factor = Outstanding Balance √∑ Vehicle Value at Origination
